Conocer más

Objetivo

Este proyecto tiene como objetivo prevenir y mitigar las inyecciones SQL en aplicaciones web, mediante la implementación de un analizador de código fuente que permita detectar posibles vulnerabilidades en el código de las aplicaciones. El analizador de código fuente se encarga de revisar el código de las aplicaciones web en busca de posibles vulnerabilidades, como inyecciones SQL, y proporciona los resultados obtenidos
